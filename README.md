# Lab-Trend-Detection-SandBox

-A sandbox project exploring time-series laboratory trend detection using simulated data. This notebook focuses on identifying and visualizing temporal patterns and directional changes in lab-style measurements over time.

-The project is exploratory and educational in nature, inspired by concepts from coursework and the MIT AI in Healthcare program.

Key Focus:
-Preprocessing longitudinal lab time-series data

-Analyzing trends and trajectory behavior

-Visualizing temporal patterns for interpretability
